1. Веб-приложение - прикладная программа, которая работает в веб-браузере пользователя
2. Стек технолгий - набор технологий, которые используются для создания проекта
3. React - библиотека JavaScript для создания пользовательских интерфейсов
4. Backend - серверная часть продукта, которая скрыта от глаз пользователя
5. Nest.JS - фреймворк, позволяющий создавать приложение на TypeScript/JavaScript
6. Frontend - все с чем взаимодействует пользователь на сайте или в приложении
7. БД - база данных
8. СУБД - система управления базами данных
9. Node.js - среда выполнения JavaScript
10. ER-диаграмма - графическая схема, которая показывает как данные связаны друг с другом в БД
11. .gitignore - файл, указывающий какие файлы не нужно отслеживать в Git
12. asyncpg - асинхронный драйвер для работы с PostgreSQL из Python
13. ORM (объектно-реляционное отображение) - технология, которые позволяет работать с БД как с обычными объектами в коде, без написания SQL-запросов вручную
14. SQLAlchemy - ORM, который позволяет работать с БД как с Python-объектами, но при этом дает полный контроль над SQL, когда это нужно
15. Postman - комплексный инструмент для работы с API, включающий тестирование, документацию, мониторинг и командную работу
16. Insomnia - программа с открытм исходным кодом, которая нацелена на быструю отладку и тестирование API
17. CRUD-эндпоинты - эндпоинты, выполняющие основные операции с данными: создание, чтение, обновление, удаление
18. Эндпоинты - конкретный URL-адрес API, который выполняет одну определенную операцию
19. CORS - механизм безопасности, разрешающий браузеру запрашивать ресурсы с другого домена
20. HTTP-запрос - сообщение, которое клиент отправляет серверу, чтобы попросить его что-то сделать
21. fetch - Web API, позволяющий отправлять HTTP-запросы из браузера
22. axios - библиотека JavaScript для отправки HTTP-запросов
23. Токен - уникальный идентификатор, позволяющий пользоваться функциями того сервиса, у которого был взят этот токен
24. aiogram - фреймворк для создания Telegram-ботов на Python
25. callback-запросы - "обратная связь" от нажатия кнопки в Telegram-боте
26. Сквозное тестирование - вид тестирования, при котором проверяются те функции, которые будут использоваться чаще всего
